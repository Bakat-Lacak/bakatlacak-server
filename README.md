# bakatlacak-server
Node JS, Express, Sequelize

App Installation

- npx sequelize db:create
- npx sequelize db:migrate
- npx sequelize db:seed:all

Set Up .ENV
- PORT= // Your port || 7000
- DB_HOST= // host here || localhost
- DB_USERNAME=
- DB_PASSWORD=
- DB_NAME=
- SECRET_KEY=
- APP_HOST= // URL of host for Multer file management || http://localhost/

Run app
`npm run dev`
